# Vue-Element-Admin-Template

基于Vue3+Element ui的后端管理网站模板

## 技术栈

- Vue3
- Vite
- TypeScript
- Element-Plus
- Axios
- Echarts
- pinia

搭建记录：[](./docs/搭建记录.md)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
